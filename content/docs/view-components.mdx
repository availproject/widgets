## Nexus Widget — The Unified Element

<span id="view-components" className="block -mt-24 pt-24" />

**Nexus Widget** is the single component for all cross-chain flows. Install it and configure the mode you need.

```bash
npx shadcn@latest add availproject/widgets/nexus
```

- [Swap and Bridge](/docs/components/swaps): `config.mode = "swap"` — swap and bridge assets through one unified intent flow.
- [Deposit](/docs/components/deposit): `config.mode = "deposit"` — route funds into a configured protocol or app action.
- [Send](/docs/components/transfer): `config.mode = "send"` — send assets to a recipient.

---

### Legacy Elements

All legacy standalone widgets have been deprecated and removed. If you are migrating an existing project, refer to the [Migration Guide](/docs/migration-guide) on how to upgrade to Nexus Widget.
