## Nexus One — The Unified Element

<span id="view-components" className="block -mt-24 pt-24" />

**Nexus One** is the single component for all cross-chain flows. Install it and configure the mode you need.

```bash
npx shadcn@latest add @nexus-elements/nexus-one
```

- [**Nexus One**](/docs/components/nexus-one): The unified element — swap, send, and deposit in one component.

### Mode-specific documentation

- [Swap and Bridge](/docs/components/swaps): `config.mode = "swap"` — swap and bridge assets through one unified intent flow.
- [Deposit](/docs/components/deposit): `config.mode = "deposit"` — route funds into configured protocol opportunities.
- [Send](/docs/components/transfer): `config.mode = "send"` — send assets to a recipient.
