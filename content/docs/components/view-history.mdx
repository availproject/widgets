<Callout>
  **⚠️ Deprecated — Use Nexus Widget**

  View History has been **removed** from Nexus Elements. Intent history is not included in Nexus Widget V1.

  For programmatic access to intent history, use the Nexus SDK directly.

  [**→ View Nexus Widget documentation**](/docs/components/nexus)
</Callout>

## Migration

Remove `ViewHistory` from your app. For programmatic intent history access:

```ts
const intents = await sdk.getMyIntents();
```

## Install Nexus Widget

```bash
npx shadcn@latest add @avail-widgets/nexus
```

See the [Nexus Widget](/docs/components/nexus) page for the unified element documentation.
