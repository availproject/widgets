<Callout>
  **⚠️ Deprecated — Use Nexus One**

  View History has been **removed** from Nexus Elements. Intent history is not included in Nexus One V1.

  For programmatic access to intent history, use the Nexus SDK directly.

  [**→ View Nexus One documentation**](/docs/components/nexus-one)
</Callout>

## Migration

Remove `ViewHistory` from your app. For programmatic intent history access:

```ts
const intents = await sdk.getMyIntents();
```

## Install Nexus One

```bash
npx shadcn@latest add @nexus-elements/nexus-one
```

See the [Nexus One](/docs/components/nexus-one) page for the unified element documentation.
