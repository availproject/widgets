<Callout>
  **⚠️ Deprecated — Use Nexus Widget**

  Fast Bridge has been **removed** from Nexus Elements. All cross-chain bridging is now handled by **Nexus Widget** with `config.mode = "swap"`. Nexus Widget automatically resolves the best route — including direct bridge paths — based on your source and destination selection.

  [**→ View Nexus Widget documentation**](/docs/components/nexus)
</Callout>

## Migration

Replace any `FastBridge` usage with `NexusWidget`:

```bash
npx shadcn@latest add @avail-widgets/nexus
```

```tsx showLineNumbers
import { NexusWidget } from "@/components/nexus/nexus";

// Bridge USDC from Arbitrum to Base — Nexus Widget resolves the optimal route
<NexusWidget
  config={{
    mode: "swap",
    prefill: {
      source: {
        token: "0xaf88d065e77c8cC2239327C5EDb3A432268e5831",
        chain: 42161,
      },
      destination: {
        token: "0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913",
        chain: 8453,
      },
    },
  }}
  connectedAddress={address}
/>
```

See the [Swap and Bridge](/docs/components/swaps) page for full configuration details.
