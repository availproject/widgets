<Callout>
  **⚠️ Deprecated — Use Nexus Widget**

Unified Balance has been **removed** from Avail Widgets. **Nexus Widget** includes inline balance views as part of its swap, send, and deposit flows.

[**→ View Nexus Widget documentation**](/docs/components/nexus)

</Callout>

## Migration

Remove `UnifiedBalance` from your app. Nexus Widget shows relevant balances inline during flow execution.

```bash
npx shadcn@latest add availproject/widgets/nexus
```

For programmatic balance access, use the Nexus SDK directly:

```ts
// Bridge balances
const bridgeBalances = await sdk.getBalancesForBridge();

// Swap balances
const swapBalances = await sdk.getBalancesForSwap();

// Format for display
const formatted = sdk.utils.formatTokenBalance(balance);
```

See the [Nexus Widget](/docs/components/nexus) page for the unified element documentation.
