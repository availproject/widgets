<Callout>
  **⚠️ Deprecated — Use Nexus One**

  Unified Balance has been **removed** from Nexus Elements. **Nexus One** includes inline balance views as part of its swap, send, and deposit flows.

  [**→ View Nexus One documentation**](/docs/components/nexus-one)
</Callout>

## Migration

Remove `UnifiedBalance` from your app. Nexus One shows relevant balances inline during flow execution.

```bash
npx shadcn@latest add @nexus-elements/nexus-one
```

For programmatic balance access, use the Nexus SDK directly:

```ts
// Bridge balances
const bridgeBalances = await sdk.getBalancesForBridge();

// Swap balances
const swapBalances = await sdk.getBalancesForSwap();

// Format for display
const formatted = sdk.utils.formatTokenBalance(balance);
```

See the [Nexus One](/docs/components/nexus-one) page for the unified element documentation.
