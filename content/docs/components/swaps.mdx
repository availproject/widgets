```bash
npx shadcn@latest add @nexus-elements/nexus-one
```

## Usage

Wrap your app with the Nexus provider first. See [Get Started](/docs/get-started) for provider setup.

```tsx showLineNumbers
import { NexusOne } from "@/components/nexus-one/nexus-one";

const USDC_ARBITRUM = "0xaf88d065e77c8cC2239327C5EDb3A432268e5831";
const USDC_BASE = "0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913";

export function SwapAndBridgeExample({
  address,
}: {
  address?: `0x${string}`;
}) {
  return (
    <NexusOne
      connectedAddress={address}
      config={{
        mode: "swap",
        prefill: {
          amount: "1",
          source: {
            token: USDC_ARBITRUM,
            chain: 42161,
          },
          destination: {
            token: USDC_BASE,
            chain: 8453,
          },
        },
      }}
    />
  );
}
```

## Configuration

Swap and Bridge uses only `mode: "swap"`. This mode is exact-in: users enter the source amount, the quote determines the receive amount, and the flow submits through `swapWithExactIn`.

| Prop | Required | Description |
| --- | --- | --- |
| `config.mode` | Yes | Must be `"swap"`. |
| `config.prefill.amount` | Optional | Prefills the source amount. |
| `config.prefill.source` | Optional | Preselects the source token and chain. |
| `config.prefill.destination` | Optional | Preselects the receive token and chain. |
| `config.allowedSourcePairs` | Optional | Restricts source token and chain choices. |
| `config.allowedDestinationPairs` | Optional | Restricts receive token and chain choices. |
| `connectedAddress` | Optional | Wallet address to use. If omitted, the connected wagmi account is used. |
| `embed` | Optional | Defaults to `true`. Set `false` to render as a modal surface. |
| `onStart`, `onComplete`, `onError`, `onClose` | Optional | Lifecycle callbacks for host apps. |
